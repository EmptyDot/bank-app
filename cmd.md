# bank-app


## pytest

```
# Run pytest on all files
pytest

# Run pytest on a specific file
pytest [file] 
```

 
## coverage
```
# Run coverage with pytest
coverage run -m pytest
```
Then:
```
# Get a report in the terminal
coverage report

# Generate html
coverage html
```

## black 
```
black [file]
```

## pylint
```
pylint [file]
```

## fastapi
```
# Run server
uvicorn api:app --reload

# Close the server 
ctrl + C 
```
